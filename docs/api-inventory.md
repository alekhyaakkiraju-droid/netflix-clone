The new Node.js backend and updated frontend must address:

- [ ] Replace `GET /api/authenticator/{email}/{password}` with `POST /api/v1/auth/login` (credentials in body)

- [ ] Add error handling (`.catch` / `try/catch`) to all 27 fetch calls
- [ ] Replace 10ms polling intervals with React state / event-driven updates
- [ ] Centralize base URL via environment variable (`VITE_API_BASE_URL`)
